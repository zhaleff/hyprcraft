## Keybinds 

|  Keys |  Action |
| :------------------------------ | :---------------------------- |
| `⎈ SUPER` + `D` | Open Terminal |
| `⎈ SUPER` + `Q` | Kill Focused Window |
| `⎈ SUPER` + `M` | Exit Hyprland Session |
| `⎈ SUPER` + `L` | Lock Screen |
| `⎈ SUPER` + `⇧ Shift` + `F` | Toggle Fullscreen |
